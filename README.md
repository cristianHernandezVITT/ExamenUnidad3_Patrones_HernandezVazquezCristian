# ExamenUnidad3_Patrones_HernandezVazquezCristian
Sistema de tareas con prioridades decoradas

Alumno: Hernández VAzquez Cristian

Proyecto: Sistema de tareas con prioridades decoradas

Funcionamiento:
El programa muestra un calendario (por defecto del mes y el año actual). PAra iniciar a usarlo debe presionar cualquier casilla del calendario. Una vez hecho, se abrirá un panel lateral que le permitirá configurar la tarea a su gusto. Es necesario que se indique título y hora. Para guardar la tarea en la fecha y hora indicada sólo presione el botón que dice guardar. A continuación puede editar esa tarea a su gusto, puede agregar un color, marcarla con prioridad alta, tacharla, borrarla, aplazarla (1 día) o cambiar su título, descripcion u hora. No olvide que para guardar los cambios es necesario presionar el botón guardar. Sin embargo, los botones que están a lado de la tarea sirven para hacer cambios rápidos.
